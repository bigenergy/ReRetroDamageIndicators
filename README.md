ReRetro Damage Indicators brings back the classic modded Damage Indicator look and feel to modern Minecraft. Ported from official mod codebase.
This mod is highly configurable and the position, opacity, size, display and more of the Retro Damage Indicator on the screen can be adjusted. This mod is client side only and does not need to be installed on the server to be used.

This mod optionaly supported Cloth Config API, install and configure config in the game.
